# Bank Loan Risk Analytics Dashboard

## Overview
An end-to-end banking analytics project that analyzes loan risk, client demographics, and repayment behavior using Power BI and Excel. Built to support data-driven credit risk assessment and financial decision-making.

## Tools & Technologies
- **Power BI** — Interactive dashboard with drill-down KPIs
- **Excel** — Data cleaning, preprocessing, and structuring
- **CSV/XLSX** — Banking client dataset with loan and demographic records

## Key Features
- Loan distribution analysis by client segment and region
- Default risk indicators and repayment behavior tracking
- Client profiling dashboard with demographic breakdowns
- Interactive slicers and filters for non-technical stakeholders

## Dataset
Contains banking client records with attributes including loan amount, client demographics, account type, and repayment status.

## Project Structure

├── Banking.csv                # Raw banking dataset
├── banking-clients.csv        # Client demographic data
